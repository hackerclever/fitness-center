วิธีการติดตั้ง
1. clone project จาก gitkraken (link:https://github.com/hackerclever/fitness-center.git)
2. เข้าถึง directory ของ project
3. สั่ง npm install
4. สั่ง composer install
5. สร้าง database ชื่อ fitness_data_center
user db = root , password db =
6. ใช้คำสั่ง php artisan migrate
7. ใช้คำสั่ง php artisan db:seed
จึงสามารถเรียกใช้ http://fitness-center.dev/ ได้

รายละเอียดการทำงาน
ระบบของเราเป็นระบบสำหรับพนักงานเพื่อลดภาระการทำงานของลูกค้า
ให้ลูกค้าสามารถดูข้อมูล Promotion, Classes, Course ได้เท่านั้น
ส่วนการจัดการสำหรับการเพิ่มข้อมูลการสมัคร Course หรือสมัครสมาชิกเป็นของทางพนักงานทั้งหมด

-Step 1:
    -สมาชิกทุกคนระดมความคิด เรื่องขอบเขตของระบบ ฟิสเนส
    -ทุกคนช่วยกันออกแบบโครงสร้าง Database โดยมีการเสนอว่าออกแบบแต่ละแบบมีข้อดีข้อเสียต่างกันอย่างไร
    -ช่วยกันแจกแจงงานออกมาเป็นส่วนๆ
    -ให้คะแนนความยากของแต่ละส่วน โดยคะแนนเต็ม 10 โดยวิธี poker
    -แบ่งงานกันโดยหารตามคะแนนความยาก
